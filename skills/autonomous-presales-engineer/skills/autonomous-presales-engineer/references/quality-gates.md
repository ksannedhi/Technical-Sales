# Quality Gates

Run these checks before final output:
1. Requirement traceability: each major RFP demand is represented.
2. BOQ fidelity: quantities and line intent are unchanged from source.
3. Scope clarity: in-scope and out-of-scope are explicit.
4. Delivery realism: timeline and dependencies are coherent.
5. Acceptance readiness: measurable acceptance criteria exist.
6. Support clarity: SLA/escalation wording is present.
7. Tone: formal reseller language, no speculative claims.
8. File completeness: all requested `.md` and `.docx` files exist.
